# Podcast de l'Associació Veïnal de Can Gaietà

## Sobre el Podcast

Podcast informatiu de l'Associació Veïnal de Can Gaietà de Tiana que ofereix una font d'informació alternativa sobre els esdeveniments del poble, discussions del ple de l'ajuntament i sessions del CAUT.

## Disclaimer Important

**Tots els podcasts (guions i audio) estan creats amb Google NotebookLM**, una eina d'intel·ligència artificial. Encara que el contingut està basat en fets reals de les fonts originals, pot contenir interpretacions o matisos que no encaixin completament amb la realitat.

**Sempre proporcionarem les fonts de dades originals** perquè qui ho desitgi pugui consultar la informació directament.

## Sobre Google NotebookLM

Google NotebookLM és una eina gratuïta disponible per a qualsevol persona amb compte de Google. Permet:
- Pujar fonts de diversos llocs, incloent vídeos de YouTube
- Demanar informació sobre les fonts
- Generar presentacions i podcasts automàticament

Els vídeos de les sessions municipals estan disponibles a YouTube i són una de les fonts principals utilitzades.
